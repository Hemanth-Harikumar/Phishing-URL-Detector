# Phishing URL Detector

A Flask-based phishing URL detection system that analyzes URLs using multiple phishing indicators and calculates a risk score.

## Features

- Risk Score (0-100)
- LOW / MEDIUM / HIGH Classification
- URL Breakdown
- IP Address Detection
- Suspicious Keyword Detection
- URL Shortener Detection
- Suspicious TLD Detection
- SQLite Database Integration
- Scan History Dashboard

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- Regex
